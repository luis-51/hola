# Mi Empresa - Página Web Profesional

Esta es una página web semiprofesional para Mi Empresa, construida con HTML, CSS y JavaScript puro.

## Características

- **Diseño Responsivo**: Adaptable a móviles y tablets.
- **Tema Oscuro/Claro**: Cambia el tema con un botón.
- **Navegación Interactiva**: Menú hamburguesa para móviles.
- **Carrusel de Imágenes**: En la galería con navegación.
- **Modal para Imágenes**: Amplía imágenes al hacer clic.
- **Formulario de Contacto**: Con validación básica.
- **Animaciones**: Fade-in en secciones al hacer scroll.
- **SEO Optimizado**: Meta tags, sitemap, robots.txt.

## Estructura de Archivos

- `index.html`: Página principal.
- `acerca.html`: Página "Acerca de".
- `servicios.html`: Página de servicios.
- `galeria.html`: Galería de imágenes.
- `contacto.html`: Formulario de contacto.
- `css/estilos.css`: Estilos CSS.
- `js/script.js`: JavaScript funcional.
- `robots.txt`: Para motores de búsqueda.
- `sitemap.xml`: Mapa del sitio.

## Cómo Usar

1. Abre `index.html` en un navegador web.
2. Navega por las páginas usando el menú.
3. Prueba el carrusel, modal y formulario.

## Tecnologías Usadas

- HTML5
- CSS3 (con variables y flexbox/grid)
- JavaScript (ES6+)
- Font Awesome (iconos)
- Google Fonts (tipografía)
